Logistic Regression Classification

Use a straight-line score when you need a category decision and a probability you can inspect.

The perceptron gave us a first classifier: score the example, guess a side, and update the boundary after mistakes. Logistic regression keeps the straight-line score, but it adds an important layer of interpretation. Instead of only saying “this side or that side,” it squeezes the score into a probability between 0 and 1.

That matters when the decision has a cost. If a moderation model is 51% sure a message needs review, you may route it differently than a message at 97%. The boundary is still linear, but the readout now says how far the example sits from that boundary in probability language. The useful question becomes: is this probability calibrated, or is it just a clean-looking number?

01

Calculate a score

Multiply features by weights, add the bias, and get one linear score z.

02

Squeeze to probability

The sigmoid function turns z into p, a number between 0 and 1.

03

Choose a threshold

Predict the positive class when the probability crosses the decision threshold.

What to watch

  • The boundary is still straight.
  • The score can become a probability instead of only a side.
  • The threshold turns probability into action.
  • Calibration asks whether probabilities match real frequencies.

How it is built

Logistic regression scores, squashes, and thresholds.

The model is linear until the final probability conversion. Training usually minimizes log loss, which strongly penalizes confident wrong probabilities.

01 · Linear score
z = w · x + b

x is the feature vector, w is the weight vector, and b is the learned bias offset.

02 · Probability
p = 1 / (1 + e-z)

The sigmoid function turns any score into a value between 0 and 1. Large positive scores move toward 1; large negative scores move toward 0.

03 · Class decision
ŷ = 1 if p ≥ t

ŷ (“y-hat”) is the predicted class. t is the threshold, often 0.5 unless the cost of mistakes suggests otherwise.

Key terms

Words that make the probability honest.

Logit
The raw linear score before the sigmoid function.
Sigmoid
An S-shaped function that maps a score to a probability-like value.
Threshold
The cutoff that turns probability into a class action.
Calibration
Whether examples predicted near 80% really happen about 80% of the time.
