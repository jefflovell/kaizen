Support Vector Classification

Find a boundary that separates classes while keeping the widest useful margin around it.

A support vector classifier is also looking for a boundary, but it is pickier than the perceptron. If many lines can separate the classes, SVC prefers the one with the widest margin: the largest useful buffer between the boundary and the closest training examples.

Those closest examples are the support vectors. They matter because they hold the boundary in place. Points far from the margin may be correctly classified, but they do not usually decide where the separating line sits. This is why SVC is a lesson in restraint: the boundary should separate the classes without balancing on every easy point.

How it is built

SVC makes the boundary less fragile.

The model balances correct classification against margin width. Soft-margin SVC accepts some mistakes when a perfect fit would be too brittle.

01 · Decision score
f(x) = w · x + b

The sign of f(x) decides the class for a linear SVC.

02 · Margin goal
maximize 2 / ||w||

A smaller weight magnitude creates a wider margin around the decision boundary.

03 · Soft margin
margin + C · violations

C controls how strongly the model punishes examples that cross or enter the margin.

Key terms

The vocabulary of margins.

Margin
The empty buffer between the boundary and the closest influential examples.
Support vector
A training example close enough to help define the boundary.
Soft margin
A version that allows some violations to avoid overfitting noise.
Kernel
A similarity function that can make nonlinear separation possible.
