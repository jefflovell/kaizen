Linear Regression

Linear regression learns a straight-line relationship between a numeric input and a numeric outcome.

Regression is the branch of supervised learning that predicts a number rather than a category. Here, the input is a GPU's gaming performance and the outcome is its used-market price.

A fitted line gives us a useful baseline: what would a GPU normally cost at this performance level? Real listings wander above and below it because price also reflects age, VRAM, condition, efficiency, scarcity, and the occasional wildly optimistic seller.

From benchmarks to a feature

Many frame rates become one comparable number.

Raw evidence might start as average frames per second across several games tested at the same resolution and graphics settings. Because one game may favor a particular GPU, a benchmark suite combines results from many games instead of trusting a single frame-rate result.

We normalize those results against a reference GPU. In this simplified example, the reference card receives a relative performance score of 100. A card that is roughly 50% faster across the benchmark suite would score near 150.

That single score becomes the model's input feature, while used price is the target it tries to predict. This index is a teaching abstraction, not an industry standard. A real pricing model might retain game-specific results alongside VRAM, age, condition, efficiency, and warranty instead of compressing performance into one number.

How it is built

The line is an equation trained to miss by less.

Linear regression begins with a weighted equation, measures how wrong its predictions are, then adjusts the weights to minimize the total squared error.

01 · Make a prediction
ŷ = b0 + Σj=1mbjxj

Σ (“capital sigma”) means “sum.” Multiply each feature xj (“x sub j”) by coefficient bj (“b sub j”), add intercept b0 (“b sub zero”), and produce estimate ŷ (“y-hat”).

02 · Measure each miss
ei = yi − ŷi

The residual ei (“e sub i”) is the vertical gap between actual sale price yi (“y sub i”) and estimate ŷi (“y-hat sub i”).

03 · Fit the coefficients
MSE = 1n Σi=1n(yi − ŷi)2

MSE means Mean Squared Error. Choose the coefficients that make this loss as small as possible. n is the number of examples, and Σ (“capital sigma”) adds their squared errors.

Key terms

The vocabulary of a fitted line.

The equation is compact; these terms describe the jobs performed by each part of it.

Coefficient
A learned weight such as bj (“b sub j”) that scales one feature.
Slope
How much the prediction changes when an input increases by one unit.
Intercept
b0 (“b sub zero”), the baseline prediction when all inputs are zero.
Residual
ei (“e sub i”), the actual value minus the prediction for example i.
Loss function
A single score that tells training how wrong the model is overall.
MSE
Mean Squared Error: the average of all squared residuals.
Multiple regression
Linear regression using several input features instead of only one.
Outlier
An unusual example whose large residual can pull the fitted line away from the broader trend.
