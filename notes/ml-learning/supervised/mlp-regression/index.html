Multilayer Perceptron Regression

A network of simple weighted units learns nonlinear interactions that one straight line cannot express.

How much of a newly released streaming title will viewers finish during its first week? Audience affinity matters—but its effect may change with awareness, release timing, runtime, tone, and social momentum.

A multilayer perceptron (MLP) passes those features through hidden neurons. Each neuron learns a small transformation; together they can represent interactions that a single linear equation misses.

How it is built

Combine. Activate. Learn backward.

The lab uses one hidden layer and squared-error loss. Every visible prediction comes from these operations.

01 · Combine the inputs
z = Σ(wixi) + b

Σ (“capital sigma”) means add the terms. wi (“w sub i”) is a weight, xi is input i, and b is the bias.

02 · Apply an activation
a = ReLU(z) = max(0, z)

a is the neuron’s activation. ReLU (Rectified Linear Unit) keeps positive values and turns negative values into zero.

03 · Update a parameter
w ← w − η(∂L/∂w)

η (“eta”) is the learning rate. ∂L/∂w (“partial L over partial w”) measures how changing this weight changes the loss.

Compact layer notationa = f(Wx + b)

This performs the same weighted-sum-and-activation calculation for every hidden neuron at once. Matrix notation compresses the scalar process—it does not replace it.

Key terms

The vocabulary of learned representations.

A network is built from simple operations; the depth comes from composing and training them together.

Hidden layer
Neurons between the raw inputs and final output that learn intermediate features.
Weight
A learned multiplier controlling the strength and direction of a connection.
Bias
A learned offset that lets a neuron shift when it becomes active.
Forward pass
Calculating activations from the inputs through to the prediction.
Backpropagation
Efficiently assigning loss responsibility backward through the network.
Gradient
The direction and sensitivity of change in the loss.
Epoch
One training pass through every example in the training set.
Hyperparameter
A setting chosen outside training, such as learning rate or hidden-neuron count.
