Start with real-world features, turn them into numbers, and assemble one complete prediction from the inside out.

A neural network diagram can look like a mysterious web of circles and lines. It becomes much simpler when we build it one operation at a time.

The same object will grow throughout this lesson: features become a vector, the vector enters a neuron, bias and activation shape its response, and several neurons become a feedforward network.

What to discover

  • Changing a feature changes the vector before it changes the network.
  • A weight controls both direction and strength: positive supports, negative opposes.
  • Bias is a learned offset, not another input feature.
  • Activation functions transform the weighted sum after bias is added.
  • A hidden layer repeats the same neuron calculation with different parameters.
  • Feedforward means using current parameters to calculate a prediction—not training them.
