Supervised Learning

Supervised learning practices with examples that already include the answer, then predicts answers for new examples.

Each training example pairs a set of input features with a known target. The learning algorithm compares its prediction with that target and adjusts the model to reduce future errors.

The first question is not which algorithm to choose. It is what kind of answer the system must produce: a category or a number.

01

Read labeled examples

Each example includes features and a known target answer.

02

Compare predictions

The model measures how its guess differs from the target.

03

Generalize

The trained model predicts answers for examples it has not seen.

Two foundational tasks: classification predicts a category; regression predicts a number.

A useful distinction

The task and the algorithm are not the same thing.

Classification and regression describe the kind of answer wanted. Algorithms describe how a model learns that answer. Trees, nearest neighbors, support vector machines, and neural networks can often be adapted to either task.
