Inspiration & purpose

From tutorial exercise to color-training arcade.

Chromatic Lab rebuilds an RGB guessing game I first made during the Colt Steele Web Developer Bootcamp. The original taught the relationship between numerical RGB values and visible color.

This version turns that lesson into a replayable training game with increasingly similar choices, timed modes, scoring, streaks, lives, and channel-by-channel feedback after every guess.

Why this stack

Browser-native by design.

The game is mostly state, timing, color math, and responsive controls, so a large framework would add more weight than value. Plain JavaScript keeps the game loop direct and easy to reason about.

Vite provides the development and deployment tooling while the browser handles the interface, keyboard, pointer, and touchscreen input. The result is small, fast, and deploys cleanly to GitHub Pages.

The complete game and its tests are available in the repository.
